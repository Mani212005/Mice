CONFIRMATION LADDER

Guidance is automatic. Actions are not.

flowchart TD
  A[Palette command] --> B[Model creates advisory plan]
  B --> C{Person accepts?}
  C -- Revise / cancel --> A
  C -- Accept --> D[Guide panel: highlight + explain]
  D --> E{Person requests “Do it”?}
  E -- No --> F[Person acts; presses Next]
  E -- Yes --> G[Show target, label, and exact proposed action]
  G --> H{Person confirms?}
  H -- No --> D
  H -- Yes, only safe verified action --> I[AXI performs one action]
  I --> D
