The monitor is intentionally operational: it shows task ownership, lifecycle truth, observed path counts, active conflicts, and one reversible next step—never a transcript or a hidden merge.

What is now explicit

  • Planner origin is visible, including safe Markdown fallback.
  • Verified is different from a process merely exiting.
  • Agents can pull bounded mission context through MCP.

What stays out of view

  • Agent transcripts, credentials, captures, process IDs, and worktree locations.
  • Automatic merge, rebase, terminal injection, or surprise reassignment.
  • False confidence when file scope is unknown.
